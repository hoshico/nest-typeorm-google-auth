## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

nest-app配下で

```bash
$ npm install
```

## Running the app

Dockerので起動は

```bash
# watch mode
$ docker compose exec node npm run start:dev

```

ローカルでの起動はnest-app配下で
```bash
# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
